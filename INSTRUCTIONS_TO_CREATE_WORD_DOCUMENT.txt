HOW TO CREATE THE WORD DOCUMENT
================================

The Python script "convert_to_word.py" was created to generate the Word document.

TO RUN THE SCRIPT, FOLLOW THESE STEPS:

OPTION 1: Using Command Prompt (Easiest)
==========================================

1. Open Command Prompt (Win + R, type "cmd", press Enter)

2. Navigate to the project folder:
   cd "C:\Users\ADMIN\Desktop\XAMPP 2025\htdocs\Peer-Peer M-Pesa debt management"

3. Make sure python-docx is installed:
   pip install python-docx

4. Run the conversion script:
   python convert_to_word.py

5. The file "P2P_DEBT_MANAGEMENT_SYSTEM_PROPOSAL.docx" will be created in the same folder

==========================================

OPTION 2: Using PowerShell (If Command Prompt doesn't work)
============================================================

1. Open PowerShell (Win + X, select Windows PowerShell Administrator)

2. Navigate to the folder:
   cd "C:\Users\ADMIN\Desktop\XAMPP 2025\htdocs\Peer-Peer M-Pesa debt management"

3. Install python-docx:
   pip install python-docx

4. Run the script:
   python convert_to_word.py

==========================================

OPTION 3: Manual Copy-Paste (If Python doesn't work)
=====================================================

1. Open the file: P2P_DEBT_MANAGEMENT_SYSTEM_PROPOSAL.md (with any text editor)

2. Select all text (Ctrl + A) and Copy (Ctrl + C)

3. Open Microsoft Word

4. Paste the text (Ctrl + V)

5. Go to Home > Styles and apply formatting:
   - Title (size 28, bold, blue)
   - Headings 1-3 (outline style)
   - Normal paragraphs

6. Save as: P2P_DEBT_MANAGEMENT_SYSTEM_PROPOSAL.docx

==========================================

OPTION 4: Using Pandoc (If you have it installed)
==================================================

1. Open Command Prompt

2. Navigate to the folder:
   cd "C:\Users\ADMIN\Desktop\XAMPP 2025\htdocs\Peer-Peer M-Pesa debt management"

3. Run:
   pandoc P2P_DEBT_MANAGEMENT_SYSTEM_PROPOSAL.md -o P2P_DEBT_MANAGEMENT_SYSTEM_PROPOSAL.docx

The Word document will be created automatically.

==========================================

RECOMMENDED: Option 1 or 4 (Automated)
      Time needed: ~5 minutes
      Quality: Professional formatting
