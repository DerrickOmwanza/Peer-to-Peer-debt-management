================================================================================
⚠️ COMMON MISTAKES TO AVOID
================================================================================

❌ WRONG API URL
   Use: http://10.0.2.2:5000/ for emulator
   NOT: http://localhost:5000/

❌ BACKEND NOT RUNNING
   Risk Score and Dispute won't work unless backend is on
   Start backend: npm start port 5000

❌ GRADLE SYNC FAILS
   Solution: File → Invalidate Caches → Restart

❌ Activity not registered
   Check: AndroidManifest.xml has the activity declared

❌ Missing imports in .kt files
   Solution: Alt+Enter auto-imports in Android Studio
